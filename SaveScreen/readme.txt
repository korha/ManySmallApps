Make screenshots

Run parameter:
Mfilepath

First symbol 'M' - mode, 'f' - fullscreen, 'w' - foreground window, 'a' - client area of foreground window (may get unexpected results).
Starting with the second character - a path to save the file, with last symbols contains "yyyyMMddhhmmsszzz.bmp"

E.g.:
fC:\MyScreens\prefix_yyyyMMddhhmmsszzz.bmp - save fullscreen screenshot to C:\MyScreens\prefix_20150102030405060.bmp file.
Failed if folder doesn't exists.
